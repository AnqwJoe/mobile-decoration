# mobile-decoration
移动端装修网站整站，采用REM适配，兼容多种尺寸设备与主流浏览器

**核心页面：**

- 首页
- 装修百科
- 装修风格
- 设计团队
- 品牌介绍
- 视频展示
- 视频详情
- 留言建议
- 装修报价

---


预览地址（**需使用手机访问，PC端建议开启chrome调试模式**）：
